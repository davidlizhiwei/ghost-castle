# 👻 幽灵古堡探险 🏰

一个适合儿童的小恐怖冒险游戏，在神秘的古堡中寻找魔法水晶并逃脱！

## 🎮 游戏说明

**目标**: 找到 5 个魔法水晶，解开谜题，逃离幽灵古堡！

**操作**:
- 使用手电筒探索各个房间
- 与物品互动获取线索和道具
- 躲避幽灵的追击
- 收集所有水晶后找到出口

## 🎯 游戏特色

- 🔦 手电筒照明系统
- 👻 随机出现的幽灵
- 🧩 解谜元素
- 💎 收集系统
- 🎨 精美的视觉效果

## 🚀 快速开始

### 本地运行

直接用浏览器打开 `index.html` 文件即可开始游戏！

或者使用 Python 启动本地服务器：

```bash
python3 -m http.server 8080
```

然后在浏览器访问：http://localhost:8080

### GitHub Pages

如果已部署到 GitHub Pages，访问：https://你的用户名.github.io/ghost-castle/

## 🎮 游戏技巧

1. 💡 **多探索**: 每个房间都可能有隐藏的物品
2. 🔑 **收集钥匙**: 有些门需要钥匙才能打开
3. 🪞 **注意镜子**: 镜子可能会揭示秘密
4. 💀 **和骷髅对话**: 它会给你重要提示
5. 👻 **躲避幽灵**: 被幽灵抓住会损失生命值

## ⚠️ 注意事项

- 游戏包含轻微的恐怖元素，适合 8 岁以上儿童
- 建议佩戴耳机以获得更好的游戏体验
- 如果感到害怕，可以随时退出游戏
